# Hackathon Ideas — Batch 2
**Hackathon:** Technotarang, Poornima University, Jaipur
**Date:** April 18, 2026
**Themes covered:** Cybersecurity, Heritage, Tourism, Web 3.0, Robotics & Drones, Space & Energy

## IDEA 7 — Cybersecurity
### "Suraksha" (Protection)

**Problem:** Rural first-time smartphone users have no defense against UPI scams, fake bank calls, and fraud apps. India lost ₹1,750 crore to cyber fraud in 2023 — 60%+ victims are rural/semi-urban.

**Solution:** Real-time scam detection + fraud simulation trainer for first-generation digital India

**Core Flow:** Scam call detected in real-time → alert user → simulate fake scam in safe environment → user learns by experiencing → family alerts for vulnerable members

**Features:**
| Feature | Type |
|---|---|
| Live scam call pattern detection | Core |
| Fraud simulation trainer (safe fake scam experience) | Differentiator |
| UPI ID risk checker before payment | Core |
| Family alert system for vulnerable members | Deep |
| Fake loan app detector | Deep |
| Community scam report database | Deep |
| Voice alerts in Hindi/regional | Deep |
| WhatsApp message scam checker | Deep |

**Winning angle:** Technical enough for cybersecurity judges. Emotional enough (retired teacher loses life savings) to win the room.

## IDEA 8 — Heritage
### "Sutr" (Thread — connecting past to present)

**Problem:** Rajasthan has 300+ folk art forms. Masters are dying. Their knowledge goes with them. Meanwhile artisans earn ₹200/day while their craft sells for ₹20,000 in Delhi galleries.

**Solution:** Living heritage archive + direct artisan marketplace — preserves knowledge, eliminates middlemen, grows artisan income

**Core Flow:** Artisan records technique → AI structures into searchable archive → tourists/buyers discover → purchase directly → school lesson plans generated from content

**Features:**
| Feature | Type |
|---|---|
| Artisan living archive (video + AI tagging) | Core |
| Direct-to-buyer marketplace (no middleman) | Core + Economic |
| Technique tutorial library | Deep |
| AI heritage story generator | Differentiator |
| School lesson plan library | Deep |
| Regional language throughout | Deep |
| Artisan income tracker | Deep |
| QR code on craft → full story | Differentiator |

**Winning angle:** At Jaipur — judges see their own culture being saved + economic justice for artisans.

## IDEA 9 — Tourism
### "Padharo" (Welcome — Rajasthani greeting)

**Problem:** India's rural tourism is invisible. Villages near Jaipur have stunning history and authentic experiences but no way to reach travelers. TripAdvisor doesn't cover them.

**Solution:** Hidden India discovery engine with voice-first host onboarding for rural communities

**Core Flow:** Traveler searches for authentic experience → discovers off-grid villages, local food, craft workshops → books directly → host onboarded via voice (no writing needed)

**Features:**
| Feature | Type |
|---|---|
| Hidden experience discovery engine | Core |
| Voice-first host onboarding | Differentiator |
| Hyper-local itinerary builder | Core |
| Economic transparency dashboard | Differentiator |
| Verified authentic experience badge | Deep |
| Local food + craft map | Deep |
| Community review system | Deep |
| Offline guide download | Deep |

**Winning angle:** Sustainable, community-led tourism. Economic transparency makes it novel vs Airbnb clones.

## IDEA 10 — Web 3.0
### "Zameen" (Land)

**Problem:** India has 66% of land disputes globally. Corrupt patwaris alter records. Farmers sell land and buyers get cheated in the 2-year gap before official records update.

**Solution:** Blockchain land record + smart contract platform for rural India

**Core Flow:** Land transaction → both parties sign digitally → smart contract holds money → both confirm → money releases + blockchain record updates instantly

**Features:**
| Feature | Type |
|---|---|
| Blockchain land record storage | Core |
| Smart contract for transactions | Core |
| Full ownership history (transparent) | Core |
| Dispute flag + alert system | Deep |
| Digital ownership certificate (PDF + QR) | Deep |
| Bank integration for loan collateral | Deep |
| Offline verification option | Deep |
| Govt integration roadmap | Differentiator |

**Winning angle:** Real blockchain utility, not crypto speculation. Land record fraud is politically powerful and technically impressive.

## IDEA 11 — Robotics & Drones
### "Dawa Udd" (Medicine Flies)

**Problem:** Medical emergency in a village. Nearest pharmacy is 60km. Road is broken. Patient needs insulin in 2 hours. A drone could reach in 18 minutes.

**Solution:** Drone-based emergency medicine delivery platform — dispatch, tracking, hub management, cold chain support

**Core Flow:** ASHA worker requests medicine → AI triages urgency → nearest drone hub dispatched → real-time tracking → delivery confirmed

**Features:**
| Feature | Type |
|---|---|
| Emergency medicine request portal | Core |
| Drone dispatch + real-time tracking | Core |
| Hub inventory management | Core |
| AI urgency triage for dispatch priority | Deep |
| Cold chain payload flagging | Deep |
| ASHA worker integration | Deep |
| Hub placement optimizer (AI) | Differentiator |
| Post-delivery confirmation | Deep |

**Winning angle:** High drama, systems-level thinking. Demo shows dispatch flow, not the drone itself.

## IDEA 12 — Space & Energy
### "Urja Mitra" (Energy Friend)

**Problem:** 40% of rural India has unreliable electricity. Farmers run diesel pumps at ₹90/litre. Solar could save ₹15,000/year but farmers don't know ROI, can't navigate subsidies without agents taking commission.

**Solution:** Solar adoption platform — ROI calculator, subsidy navigator, peer network, carbon credit access for rural farmers

**Core Flow:** Input electricity usage + diesel costs + location → AI calculates exact savings + payback period → finds subsidies → generates application → connects to nearby solar farmers for real proof

**Features:**
| Feature | Type |
|---|---|
| Solar ROI + payback calculator | Core |
| Subsidy eligibility finder + application guide | Core |
| Peer network (farmer testimonials, real numbers) | Deep |
| Post-installation energy monitor | Deep |
| Carbon credit access module | Differentiator |
| Vendor comparison | Deep |
| Diesel vs solar live cost tracker | Differentiator |
| Village energy transition dashboard | Deep |

**Winning angle:** Carbon credit income for rural farmers is genuinely novel. Clean energy + economic uplift in one product.
