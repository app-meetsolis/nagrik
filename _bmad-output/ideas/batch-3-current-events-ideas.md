**Hackathon:** Technotarang, Poornima University, Jaipur

**Angle:** Ideas tied to current global events — wars, climate crisis, mental health epidemic, misinformation warfare, food supply disruption

## IDEA 13 — Cybersecurity
### "Sachchi Khabar" (Real News)

**Global event hook:** Every active war (Ukraine, Gaza, Sudan) is fought with deepfakes + misinformation. India's 2024 elections saw thousands of AI-generated fake political videos. WhatsApp forwards cause riots in India.

**Problem:** Village WhatsApp group gets a deepfake video of a religious/political figure. Nobody has tools to check. Tension rises within hours. No checker exists in Indian languages.

**Solution:** AI deepfake detector + fact-checker + WhatsApp-native misinformation shield

**Core Flow:** Forward suspicious content → AI detects if fake → fact-checks claims → shows source credibility history → community flag system alerts others

**Features:**
| Feature | Type |
|---|---|
| Deepfake video/image/audio detector | Core |
| Text claim fact-checker | Core |
| WhatsApp-native flow | Differentiator |
| Source credibility history score | Deep |
| Community flag + mass alert | Deep |
| Hindi/regional language output | Deep |
| Election-mode heightened sensitivity | Differentiator |
| "How it was faked" media literacy explainer | Deep |

**Winning angle:** Every judge has received a fake WhatsApp forward. Most personally relatable problem in the room.

## IDEA 14 — Clean & Green
### "Jal Rakshak" (Water Guardian)

**Global event hook:** UN declared water scarcity a global security threat. Rajasthan had its worst groundwater depletion report ever. Wars in the Middle East are increasingly fought over water access.

**Problem:** Village borewell has no monitoring. Farmers pump without limit. By May, it's dry. Community suffers 3 months. With data, they would have managed it differently.

**Solution:** Community water monitoring + AI budget + rain harvesting planner + conflict mediator

**Core Flow:** Track water table → AI projects depletion date → recommends conservation → mediates fair allocation between farmers → finds govt water schemes

**Features:**
| Feature | Type |
|---|---|
| Water table level tracker | Core |
| AI depletion projection | Core |
| Community usage dashboard | Core |
| Rain harvesting site recommender | Differentiator |
| Fair water allocation calculator | Differentiator |
| Govt water scheme finder | Deep |
| Drought early warning | Deep |
| Village water health score (public) | Deep |

**Winning angle:** "Rajasthan is facing its worst groundwater crisis. This platform would have warned them 6 weeks before the well ran dry."

## IDEA 15 — Healthcare
### "Mann Mitra" (Mind Friend)

**Global event hook:** WHO declared global mental health epidemic. India: 13,000+ student suicides in 2023 (NCRB). JEE/NEET pressure cooker is creating a mental health crisis in tier-2/3 India.

**Problem:** 18-year-old in Kota, failed NEET 3rd time. Can't tell parents. No therapist for 200km. No money for sessions. No one at 2am when it gets bad.

**Solution:** 24/7 AI mental health companion — India-context trained, Hindi-first, crisis-aware, with peer circles

**Core Flow:** Daily 2-min check-in → mood pattern detection → AI conversation → crisis flag → auto-escalation to free Indian helplines → peer circle connection

**Features:**
| Feature | Type |
|---|---|
| 24/7 AI companion (India-context trained) | Core |
| Hindi + regional language | Core |
| Mood tracking + pattern detection | Deep |
| Crisis detection + helpline escalation | Core (safety) |
| Anonymous peer circles by situation | Deep |
| Family understanding module | Differentiator |
| Daily micro-interventions | Deep |
| Progress tracker | Deep |

**Winning angle:** Student suicide statistics are devastating. Healthcare judge with children will feel this deeply.

## IDEA 16 — Sustainability & Development
### "Fasal Badlo" (Change Your Crop)

**Global event hook:** Russia-Ukraine war disrupted global wheat supply. Climate change is shifting monsoon patterns — Rajasthan farmers growing wheat for 40 years seeing 30% yield drop.

**Problem:** Farmer in Pali grew wheat his whole life. Last 3 seasons: terrible. He doesn't know why, doesn't know what else to grow. Neighbor switched to cumin 2 years ago — earning double. Farmer found out last week at the market.

**Solution:** Climate-adaptive crop intelligence + transition roadmap + market demand forecasting

**Core Flow:** Input location + current crops → AI analyzes 10-year rainfall trend shifts → recommends climate-adapted crops → shows neighbor proof → generates 3-season transition plan → alerts on market demand + global supply spikes

**Features:**
| Feature | Type |
|---|---|
| Climate-adaptive crop recommendation | Core |
| District-level rainfall trend analysis | Core |
| Neighbor proof / peer adoption data | Differentiator |
| 3-season transition roadmap | Deep |
| Market demand forecaster | Deep |
| Global supply disruption alerts | Differentiator |
| Soil + new crop compatibility | Deep |
| Income comparison visual | Deep |

**Winning angle:** Connects global war-disrupted food supply to a Rajasthani farmer's income. Macro-to-micro storytelling.

## IDEA 17 — Ecosystems / Web 3.0
### "Sach Chain" (Truth Chain)

**Global event hook:** Governments worldwide weaponize information. Post-disaster in India: 40 fake donation links go viral within hours. Fake govt orders cause panic. Real emergency info gets buried.

**Problem:** Cyclone hits. Fake bank accounts for "donations" circulate. Fake evacuation orders spread panic. People can't tell what's real. Money goes to scammers, not victims.

**Solution:** Blockchain-verified emergency broadcast system + disaster donation verifier + fake govt order detector

**Core Flow:** Only verified officials can post → cryptographically signed → citizens verify any govt order/donation link → community trust voting → automatic warning labels on flagged content

**Features:**
| Feature | Type |
|---|---|
| Blockchain-signed official broadcasts | Core |
| Donation link verification | Core |
| Fake govt order detector | Core |
| Community trust voting | Deep |
| Official source registry (blockchain) | Deep |
| Real-time disaster information hub | Deep |
| WhatsApp integration | Differentiator |
| Media organization verification API | Differentiator |

**Winning angle:** Combines Web3 real utility + disaster response + India-specific context. Not crypto speculation — actual life-saving utility.
