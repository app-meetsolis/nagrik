# Hackathon Ideas — Batch 1
**Hackathon:** Technotarang, Poornima University, Jaipur
**Date:** April 18, 2026
**Team:** Harigopal (AI builder) + 2 (design/research/pitch)
**Context:** Ideas generated from Hari's lived experience — rural Rajasthan, healthcare gaps, education system failures, civic accountability breakdown, rural-to-city transition

---

## IDEA 1 — Healthcare
### "Swasth Setu" (Health Bridge)

**Problem:** Rural patient has symptoms at 11pm. Nearest doctor is 100km away. They either ignore it (gets worse) or travel in panic (expensive, exhausting).

**Solution:** AI-powered rural health triage + telemedicine coordination platform

**Core Flow:** Describe symptoms in Hindi/Rajasthani → AI triages severity → home care / PHC video call / emergency hospital routing

**Features:**
| Feature | Type |
|---|---|
| Symptom AI triage (3 severity levels) | Core |
| Nearest hospital finder by specialty + cost estimate | Core |
| ASHA worker community health dashboard | Deep |
| Family health wallet (digital medical records) | Deep |
| Medicine availability in local pharmacies | Deep |
| Emergency SOS → family + ambulance | Deep |
| Post-visit summary generator | Deep |
| Offline/2G capable | Differentiator |

**Winning angle:** "A child dies on a 100km trip to the hospital because no one knew it was an emergency." Judges don't forget that story.

## IDEA 2 — EdTech
### "Udaan" (Taking Flight)

**Problem:** 18-year-old in Pali, Rajasthan thinks his only options are NEET, JEE, or UPSC. Nobody told him 50+ other careers exist and pay better.

**Solution:** Career intelligence + life readiness platform for tier-2/3 India students

**Core Flow:** 3-minute quiz (interests, skills, location, family income) → 50+ career paths with realistic roadmaps from tier-3 towns

**Features:**
| Feature | Type |
|---|---|
| Career discovery quiz (not generic Myers-Briggs) | Core |
| 50+ career paths with tier-3 town roadmaps | Core |
| Success stories from similar backgrounds | Deep |
| City survival guide (metro, documents, hospitals, scams) | Deep |
| Scholarship finder by state + category | Deep |
| Life skills short modules (communication, money, confidence) | Deep |
| Peer community by town/district | Deep |
| Parent explainer mode (help convince parents) | Differentiator |

**Winning angle:** Every judge in that room knows a kid who wasted 3 years on NEET. This is personal to them.

## IDEA 3 — FinTech
### "Paisa Samjho" (Understand Money)

**Problem:** Rural families keep ₹5 lakh in FDs losing to inflation. They fear investing. The ones who do invest get scammed by chit funds.

**Solution:** Visual financial literacy + safe micro-investment platform for rural India

**Core Flow:** Input current savings approach → see exactly what you're losing vs gaining → learn alternatives in simple Hindi → start micro-SIP from ₹100

**Features:**
| Feature | Type |
|---|---|
| Savings erosion calculator (inflation impact) | Core |
| FD vs SIP vs Gold visual simulator | Core |
| Risk profiler in simple village language | Core |
| Chit fund / investment scam detector | Differentiator |
| Goal-based savings (wedding, house, emergency fund) | Deep |
| Insurance basics explainer | Deep |
| Gamified financial literacy (badges, streaks) | Deep |
| Family financial health score | Deep |

**Winning angle:** "75% of rural Indians have never invested outside FDs/gold." FinTech judges who know India's savings crisis will feel this.

## IDEA 4 — Clean & Green / Ecosystems
### "Nagrik" (The Citizen)

**Problem:** Garbage on the road. Police standing 2 meters away. Nobody says anything. The ward officer hasn't visited in 6 months. No accountability because nothing is visible.

**Solution:** Gamified civic accountability platform — public shame + community recognition engine

**Core Flow:** Citizen photos civic issue + GPS → AI verifies + routes to authority → 48hr clock starts → if unresolved, auto-escalates → public dashboard shows everyone's score

**Features:**
| Feature | Type |
|---|---|
| Photo + GPS issue reporting | Core |
| AI issue categorization + authority routing | Core |
| 48hr auto-escalation engine | Core |
| Public civic health dashboard (ward-level) | Differentiator |
| Authority resolution rate visible to public | Differentiator |
| Community hero points + leaderboard | Deep |
| Before/after photo proof for resolved issues | Deep |
| Monthly ward report card | Deep |

**Winning angle:** Most visceral demo — judges who've complained about garbage and been ignored will fund this emotionally.

## IDEA 5 — Transportation
### "Pehli Baar" (First Time)

**Problem:** First time in a city from a village — metro, autos, hospital OPD, government offices. Millions feel lost, confused, and exploited every day.

**Solution:** City survival guide for first-time urban visitors from rural India — purpose-specific, offline-capable

**Core Flow:** Select city + purpose (hospital / govt office / job interview / college) → get step-by-step survival guide in local language, including scam alerts and what documents to carry

**Features:**
| Feature | Type |
|---|---|
| Purpose + city → complete guide | Core |
| Hospital navigation (OPD, emergency, specific dept) | Core |
| Government office procedure guides | Core |
| Transport route explainer (metro, bus, auto) | Core |
| Scam alert database by city | Differentiator |
| Documents checklist by purpose | Deep |
| Offline download for full city guide | Deep |
| SOS + "I'm lost" location share | Deep |
| City etiquette guide | Deep |

**Winning angle:** Highly unique — nobody has built this. Every judge has a relative who got lost or scammed on their first city trip.

## IDEA 6 — Sustainability & Development
### "Khet Budi" (Farm Intelligence)

**Problem:** Farmer in Pali uses same seed variety his grandfather used. Nobody told him there's a drought-resistant variant yielding 40% more, subsidized by the government. He found out 3 years too late.

**Solution:** AI farming intelligence platform — crop health, subsidies, market timing, voice-first for low-literacy farmers

**Core Flow:** Input crop + location + season → personalized advice → photo of sick crop → disease detected → subsidy eligibility shown → nearby mandi prices with sell timing

**Features:**
| Feature | Type |
|---|---|
| Crop disease detection via photo (AI) | Core |
| Personalized farming advice by crop/region/season | Core |
| Hyperlocal weather + irrigation schedule | Core |
| Subsidy eligibility finder | Differentiator |
| Multi-mandi price tracker + sell timing recommendation | Differentiator |
| Peer community by crop + district | Deep |
| New technique video library (regional language, short) | Deep |
| Voice-first interface (for low-literacy farmers) | Deep |
| Soil health tracker | Deep |

**Winning angle:** Farmer distress is India's most politically sensitive topic. Sustainability judges see impact + technical innovation together.
